# Introducción a CSS

CSS (Cascading Style Sheets) u "Hojas de Estilo en Cascada" es el lenguaje que utilizamos para dar estilo y apariencia a nuestros documentos HTML.

Si HTML es el esqueleto y la estructura de una página web, **CSS es la piel**, el maquillaje y la ropa. Se encarga de todo lo visual: colores, tipografías, espaciado, posicionamiento y, muy importante, cómo se adapta la página a diferentes tamaños de pantalla (Diseño Responsivo).

La filosofía de estos apuntes es cubrir los conceptos **esenciales** de CSS de forma concisa, centrándonos en entender el "por qué" de su funcionamiento para construir una base sólida.
